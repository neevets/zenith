# Zenith Language Support for VS Code

Zenith is a high-performance programming language that transpiles to PHP 8.1+. This extension provides syntax highlighting and language support for Zenith in Visual Studio Code.

## Features

- **Syntax Highlighting**: Complete syntax highlighting for Zenith keywords, types, and operations.
- **Language Association**: Automatically recognizes `.zen` files.
- **Project Link**: Integration with Zenith compiler projects.

## Usage

Simply open any `.zen` file to get started.

## More Information

For more information about Zenith, visit [zenithlang.xyz](https://zenithlang.xyz).
